# piRSSonite Application Overview

## Functionality
piRSSonite is a full-featured, self-hosted RSS aggregator that allows users to centralize their news consumption. Its core capabilities include:
*   **Feed Management:** Users can subscribe to standard RSS/Atom feeds and organize them into hierarchical folders.
*   **Organization:** Supports drag-and-drop reordering for both individual feeds and folders, allowing for a highly non-rigid organization structure.
*   **Real-time Updates:** Includes an auto-refresh mechanism and WebSub support for near-instant updates from compatible sources.
*   **Read Status Tracking:** Automatically tracks read/unread status for articles, providing visual counts at both the feed and folder level.
*   **Data Portability:** Supports OPML import and export, ensuring users can migrate their subscriptions to and from other readers.
*   **View Options:** Offers filtered views to hide empty feeds and a global "All Items" stream.

## Visual Design & User Interface
The application features a distinct, premium visual style built around a "Glassy Morphism" theme.
*   **Aesthetic:** A dark, immersive interface using deep blue-grey backgrounds with semi-transparent, glass-like cards. It heavily utilizes monospace typography (Courier New, Fira Code) to give it a technical, precision-engineered feel.
*   **Color Palette:**
    *   **Backgrounds:** Deep dark blues (`#2a2e3e`) and translucent overlays.
    *   **Accents:** High-contrast Gold (`#FFD700`) for unread indicators, Cyan (`#4AADC1`) for interactive elements, and Green (`#1ECB7F`) for success/read states.
*   **Layout:**
    *   **Sidebar:** A resizeable left-hand panel displaying the feed hierarchy, unread counts, and settings access.
    *   **Main Content:** A larger right-hand pane for scrolling through articles, likely presenting them in a list or card format with rich previews.
*   **Experience:** The UI emphasizes fluidity with backdrop blurs, hover effects, and smooth transitions, creating a polished and responsive feel.
